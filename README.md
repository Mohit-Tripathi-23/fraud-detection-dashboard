# AI-Based Fraud Detection and Risk Monitoring Dashboard

This project is a Streamlit-based machine learning dashboard that simulates fraud risk detection for banking transactions.

## Features

- Single transaction fraud assessment
- Batch CSV upload
- Fraud analytics dashboard
- Risk classification:
  - Low Risk
  - Medium Risk
  - High Risk

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn

## Use Case

This project demonstrates how banks can use AI to identify risky transactions, reduce fraud exposure, and support manual review workflows.

## Installation

```bash
git clone https://github.com/your-username/fraud-detection-streamlit-dashboard.git
cd fraud-detection-streamlit-dashboard
pip install -r requirements.txt
python train_model.py
